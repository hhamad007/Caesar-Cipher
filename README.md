# Caesar Cipher Encrypter

A simple web application to encrypt text using the Caesar Cipher technique.

## Features

- Encrypts plain text using a user-defined shift value.
- Automatically converts input to uppercase.
- Copy encrypted text to clipboard with a single click.
- Clean and modern UI.

## Files

- [`index.html`](index.html): Main HTML file for the app UI.
- [`index.css`](index.css): Styles for the application.
- [`index.js`](index.js): JavaScript logic for encryption and clipboard functionality.

## How It Works

The Caesar Cipher shifts each letter in the input by a specified number of positions in the alphabet. Non-alphabetic characters remain unchanged.

## Running Locally

1. Clone or download this repository.
2. Open `index.html` in your web browser.

No build steps or dependencies required.
